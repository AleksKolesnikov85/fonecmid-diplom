# Инструкция по начальной настройке

1. Создать группу и бота в Telegram. Это необходимо для отправки уведомлений специалистам при создании новой заявки (документа "Обслуживание клиента"), а также при изменении исполнителя, даты или времени проведения работ.

2. Заполнить значение констант:
- *"Абонентская плата"* и *"Работы специалиста"* – подходящие по смыслу наименования услуг из справочника "Номенклатура". В случае отстутсвия создать необходимую номенклатуру.
- *"Токен управления телеграм-ботом"* – значение токена бота — Telegram.
- *"Идентификатор"* группы для оповещения" – идентификатор созданной ранее группы Telegram.
- *"НДФЛ"* - величина текущего процента НДФЛ для расчета удержаний.

3. Заполнить данные графика работы. Для этого в справочнике "Графики работы" добавить график работы (например, "Пятидневка"). Затем в подменю Сервис подсистемы "Учет заработной платы" открыть обработку "Запонение графика". В поле Период выбрать период заполнеиня графика (год). 
В графе График выбрать добавленный ранее в справочник график ("Пятидневка"). В поле "Выходные дни" через запятую перечислить порядковый номер выходных дней (6,7 для субботы и воскресенья). Нажать кнопку "Заполнить". 
Проверить корректность заполнения графика в регистре "Графики работы".
При необходимости расчетов с использованием календарных дней, например, при расчете отпусков сотрудников, добавить календарный график в справочник и заполнить его с помощью той же обработки без указания выходных дней. 
В дальнейшем при заполнении документа "Начисление зарплаты" при выбранном виде расчета "Отпуск" следует указывать соответствующий календарный график.

4. В Справочнике "Пользователи" создать пользователя для каждого сотрудника. На вкладке "Права доступа" назначить каждому пользователю соответствующий должности профиль.
Созданные профили групп доступа "Специалист", "Менеджер", "БухгалтерИТФирмы" и "Кадровик-расчетчик" являются поставляемыми и не могут быть изменены.
Подробнее с составом ролей для этих профилей можно ознакомиться по следующему пути: "Администрирование" – "Настройки пользователей и прав" – "Профили групп доступа".

5. Внести данные о сотрудниках: 
Заполнить справочник "Сотрудники". Это также может сделать пользователь с профилем "Кадровик-расчетчик".
Заполнить регистр сведений "Условия оплаты сотрудников". В котором **обязательно** нужно указать подразделение, в котором состоит сотрудник. Так как в фирме две категории сотрудников "Специалист" и "Административный отдел", то они внесены в базу в качестве предопределенных.
Для специалистов помимо оклада дополнительно необходимо указать процент, начисляемый за выполненные работы.

### Примечания:

Все вышеперичисленные объекты доступны в подсистеме "Добавленные объекты".
Пользователю с административными правами нужно проверить включение регламентного задания "Отправка уведомлений в Телеграм". Также можно поменять настройки расписания отправки уведомлений.
Добавился новый вид договоров контрагентов "Абонентское обслуживание". При выборе данного вида договора необходимо заполнить дополнительные поля "Период действия договора", "Сумма абоненсткой платы" и "Стоимость часа".
У бухгалтера появилась возможность автоматического формирования документов по реализации за месяц при помощи обработки "Массовое создание актов" в подсистеме "Обслуживание клиентов".
Появилась возможность автоматического заполнения документа "Реализация товаров и услуг" суммой ежемесячной абонентской платы и суммой за выполненные работы при выбранном действующем договоре абонентского обслуживания.
